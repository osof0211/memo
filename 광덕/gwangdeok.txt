
git config --global user.name "Your Name Here"
물론, “Your Name Here”의 인용부호 안에 자신의 이름을 넣어야 한다.
다음엔, 당신의 이메일을 말해준다. 조금 전에 GitHub.com을 가입할 때 사용한 이메일이어야 한다. 
다음과 같이 한다:
git config --global user.email "your_email@youremail.com"
